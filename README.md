### 🩺 Matricare
Born at a GenAI Hackathon, this is an AI-powered maternal-health assistant built for rural India. It flags high-risk pregnancies early, generates nutrition plans tailored to the patient, and — the part I'm proudest of — keeps working in low-connectivity conditions instead of assuming a stable network. Regional-language support and a touch-first UI mean it's built for someone's grandmother, not just a tech reviewer.

**Impact:** Top 75, GenAI Hackathon Mumbai — Grand Finale

### 🕵️ Sherlock AI — Candidate Identifier
Built solo for an AI Engineering Challenge, this system verifies interview candidates by fusing six different evidence sources — transcripts, camera activity, speaking behavior, and simulated face/voice recognition — into one live, explainable confidence score. It ships with a Streamlit dashboard for real-time ranking and auto-generates full PDF investigation reports, so the output isn't just a number, it's a case file.

**Impact:** Modular multi-engine architecture, designed and built end-to-end alone

`Python` `Streamlit` `Matplotlib` `ReportLab` `RapidFuzz`
